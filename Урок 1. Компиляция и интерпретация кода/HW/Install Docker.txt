Установка Docker на Windows

1. Перейдите на официальный сайт Docker и нажмите на кнопку "Download Docker Desktop".
2. Загрузите установщик Docker Desktop для Windows.
3. Запустите загруженный установщик Docker Desktop.
4. В процессе установки у вас могут появиться запросы на разрешение системных изменений, 
    такие как разрешение установки гипервизора и настройку сетевых параметров. Разрешите эти запросы.
5. После завершения установки Docker Desktop будет запущен. В системном трее (рядом с часами) появится иконка Docker.
6. Щелкните правой кнопкой мыши на иконке Docker в системном трее и выберите пункт "Settings" (Настройки).
7. В настройках Docker Desktop убедитесь, что опция "Expose daemon on tcp://localhost:2375 without TLS" 
    (Предоставить доступ к демону по адресу tcp://localhost:2375 без TLS) отмечена.
8. Нажмите кнопку "Apply & Restart" (Применить и перезапустить), чтобы применить настройки и перезапустить Docker.
9. После перезапуска Docker будет готов к использованию. Вы можете открыть командную строку или PowerShell 
    и выполнить команду docker version, чтобы убедиться, что Docker установлен и работает корректно.
