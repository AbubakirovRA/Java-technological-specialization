
Выполнены основные задачи по созданию контроллеров (BookController, ReaderController, IssuerController) и сервисов (BookService, ReaderService, IssuerService) для управления книгами, читателями и выдачей.
Добавлены новые методы в IssueService.java для проверки наличия книг у читателя перед выдачей и получения списка всех выдач для читателя.
В классе Issue.java произведено разделение поля timestamp на issuedAt и returnedAt, а также добавлен запрос PUT /issue/{issueId}, закрывающий выдачу и устанавливающий дату возврата.
В MyBean.java добавлен метод-обработчик события подключения к базе данных, выводящий значение свойства maxAllowedBooks из конфигурации приложения.
В ReaderRepository.java добавлены методы для получения списка всех читателей, удаления читателя по id и сохранения нового читателя.
В IssueRepository.java реализованы методы для работы с выдачей: сохранение выдачи, получение списка выдач по читателю, получение активных выдач по читателю, получение выдачи по идентификатору, а также закрытие выдачи с установкой даты возврата.
