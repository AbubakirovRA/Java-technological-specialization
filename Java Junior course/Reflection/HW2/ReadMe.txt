В Homework.java добавлены новые аннотации: BeforeEach, AfterEach, Skip. Эти аннотации используются для определения методов, которые должны выполняться перед каждым тестом, после каждого теста и для пропуска теста соответственно. Также внесены изменения в аннотацию Test, добавлен параметр order, который используется для определения порядка выполнения тестов. Внутри класса MyTest добавлены методы, помеченные новыми аннотациями:
@BeforeEach - для метода, который должен выполняться перед каждым тестом.
@AfterEach - для метода, который должен выполняться после каждого теста.
@Skip - для метода, который не должен быть запущен во время тестирования.
Кроме того, добавлены параметры order для методов, помеченных аннотацией @Test. Эти параметры определяют порядок выполнения тестов.
В методе main добавлен вызов статического метода runTest класса TestProcessor для запуска тестов в классе MyTest.

​
В TestProcessor.java  для поддержки аннотаций BeforeEach и AfterEach внесены два новых метода: invokeBeforeMethods и invokeAfterMethods. Эти методы выполняют методы с аннотациями BeforeEach и AfterEach. Когда тест запускается, сначала выполняются методы с аннотацией BeforeEach, затем тестовый метод, и, далее, методы с аннотацией AfterEach. Это позволяет выполнять дополнительные действия до и после каждого теста.
В методе runTestMethod добавлена проверка наличия аннотации Skip для тестового метода. Если тест имеет эту аннотацию, он будет пропущен, и будет выведено сообщение в консоль.
Метод getSortedTestMethods изменен для сортировки тестовых методов по значению параметра order в аннотации Test. Тесты будут выполняться в порядке, определенном значением order. Это обеспечивает управление порядком выполнения тестов в зависимости от их приоритета.
